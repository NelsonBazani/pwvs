Resumo sobre a matéria

Fundamentos de Programação Orientada a Objetos é uma das matérias mais complexas e que necessita de uma atenção maior. Para um bom entendimento da matéria, é necessário seguir um passo a passo, pois se pular demais a matéria, pode perder conteúdo importante.

Para quem não tem conhecimento para a área, não tem com o que se preocupar, pois o curso começa do mais básico e vai avançando aos poucos para que ninguém fique para trás.

Hoje existem diversas plataformas que é possível fazer a programação, sendo algumas delas o Java, Python, Ruby, C# e entre outros. O maior Foco da nossa matéria será Java, porém, antes de iniciar os estudos em Java, usaremos o Portugol para desenvolver a logística de programação.
