O que é a Linguagem de Marcação?

Muitas pessoas confundem Linguagem de Marcação com Linguagem de Programação, porém eles tem algumas diferenças. A Linguagem de programação é construído por um conjunto de regras sintáticas e semânticas, que podem ser compilado e transformado em um programa de computador. Já a Linguagem de Marcação servem basicamente para definir como um determinado conteúdo será exibido na tela. As linguagens mais utilizadas são HTML, CSS e JavaScript, porém, na nossa escola as Linguagens de Marcação mais utilizadas serão HTML e CSS.
